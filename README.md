# Portafolio — Pensamiento de Ingeniería

## Cómo usar
1. Abre `index.html` en tu navegador.
2. Edita tu nombre y textos:
   - `index.html`
   - `tareas/tarea-01.html` ... `tareas/tarea-08.html`
3. Sustituye imágenes en `assets/img/` (mismos nombres).
4. Agrega videos en `assets/video/`:
   - `t1-demo.mp4` / `t1-demo.webm` (y así para cada tarea)

## Publicar en GitHub Pages (opcional)
- Sube esta carpeta a un repo y activa Pages desde la rama principal.
